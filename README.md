Sudoku Solver (C)

A Sudoku Solver built in C using the Backtracking Algorithm. The program automatically solves a partially completed 9×9 Sudoku puzzle by recursively filling empty cells while following all Sudoku rules.

Features
Solves standard 9×9 Sudoku puzzles
Uses the Backtracking algorithm
Validates row, column, and 3×3 subgrid constraints
Recursive and efficient implementation
Beginner-friendly and well-structured code
Technologies Used
C Programming Language
GCC Compiler
Visual Studio Code / Code::Blocks
How It Works

The solver:

Finds an empty cell.
Tries numbers from 1 to 9.
Checks whether the number is valid.
Recursively continues solving the puzzle.
Backtracks whenever a dead end is reached.
